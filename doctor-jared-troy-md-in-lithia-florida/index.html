Is my tummy tuck scar too high?
Tummy Tuck incision too high?
Oct 18, 2017
Your incision is a little higher than would be ideal. This was probably done to avoid a T type incision which can result if all of the old belly button opening isn't taken out completely. If if continues to be a problem, you can have it evaluated for a possible revision after a year. good luck.
